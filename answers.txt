Q1: Why might we consider using Git?

Answer: Git is a great way to collaborate with others on a project. It allows you to track changes to a project, and to revert to a previous version if needed.

Q2: What field could data science be applied to? Give an Example.

Answer: Data science can be applied to any field that has data. For example, data science can be used to analyze customer behavior in a retail store, or to predict the weather.
